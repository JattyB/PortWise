ProjectDiscovery Wappalyzer fingerprints
========================================

Source:
- https://github.com/projectdiscovery/wappalyzergo
- Files: `fingerprints_data.json`, `categories_data.json`

License:

The data is shipped as read-only package data so PortWise can perform native
technology fingerprinting after a plain `pip install`, without requiring a
runtime download or external binary.
